# Industrial Storage Rentals — Branding Package

## Brand Identity

### Name
**Industrial Storage Rentals** (or **ISR** for short)

### Tagline
"Professional Warehouse Space. Built for Growth."

### Brand Promise
We provide secure, professional, and flexible industrial warehouse space designed for businesses that demand quality and transparency.

---

## Visual Identity

### Color Palette

**Primary Blue**
- Hex: #0099cc
- RGB: 0, 153, 204
- Usage: Headers, CTAs, accents, primary UI elements
- Represents: Trust, professionalism, security

**Secondary Gold**
- Hex: #ffdb58
- RGB: 255, 219, 88
- Usage: Highlights, hero accents, premium features
- Represents: Value, premium quality, confidence

**Dark Gray**
- Hex: #1a1a1a
- RGB: 26, 26, 26
- Usage: Text, backgrounds, strong contrast
- Represents: Stability, professionalism

**Light Gray**
- Hex: #f8f9fa
- RGB: 248, 249, 250
- Usage: Backgrounds, cards, sections
- Represents: Clean, modern, professional

**Accent Orange** (Optional)
- Hex: #ff6b35
- Usage: Alerts, emphasis, secondary CTAs
- Represents: Energy, action, urgency

### Typography

**Headlines: Roboto / System Sans-serif**
- Weight: 900 (bold)
- Size: 3.5rem (hero), 2.5rem (section headers), 1.15rem (card headers)
- Letter-spacing: -1px to -0.5px for tightness

**Body: System Sans-serif (-apple-system, BlinkMacSystemFont, 'Segoe UI', 'Roboto')**
- Weight: 400 (regular), 600 (semi-bold)
- Size: 1rem (body), 0.95rem (secondary)
- Line-height: 1.6

**Accent: 600-700 weight for emphasis**

## Logo & Icon System

### Logo Description
- **Icon**: Simple blue warehouse/building outline (18' door style)
- **Text**: "INDUSTRIAL STORAGE RENTALS" (all caps, tight letter-spacing)
- **Format**: SVG icon + text lockup (horizontal)
- **Minimum size**: 120px width
- **Clear space**: 20px around entire logo

### Icon Usage
- SVG format for scalability
- Blue (#0099cc) on white backgrounds
- White on blue/dark backgrounds
- Gold highlights for premium/featured items

## Imagery & Photography

### Photography Style
- **Industrial warehouse interiors** (clean, well-lit)
- **Equipment and machinery** in secure storage
- **Professional operations** (loading, forklifts, organized spaces)
- **Bright, natural lighting** (no dark corners)
- **Minimal people** (focus on space/facilities)

### Hero Image Guidelines
- Full-width, high-resolution
- Professional quality (2000px+ width)
- Clean, organized warehouse environment
- Neutral color tones (grays, blues, industrial)

## Tone & Voice

### Brand Voice
- **Professional**: Industry expertise, credible, trustworthy
- **Direct**: Clear, straightforward communication
- **Confident**: Assured without being arrogant
- **Helpful**: Focused on customer needs and solutions

### Key Messages
1. "Secure, Professional Warehouse Space"
2. "Month-to-Month Flexibility"
3. "Enterprise-Grade Security"
4. "Transparent Pricing"
5. "Built for Serious Operations"

### Copy Guidelines
- Use active voice
- Short, punchy sentences for headers
- Clear benefits-focused copy
- Avoid jargon; explain features clearly
- Professional but approachable tone

## Website Design System

### Layout
- **Max-width**: 1200px
- **Padding**: 20px (mobile), 60px+ (desktop)
- **Section spacing**: 100px (desktop), 60px (mobile)

### Components

**Cards**
- Background: White (#fff) or light gray (#f8f9fa)
- Border: 1px solid #ddd
- Padding: 30px
- Border-radius: 8px
- Hover: Lift effect (translateY -8px) + shadow

**Buttons**
- Primary: Blue background, white text
- Secondary: Transparent with blue border
- Size: 12-14px padding, 16px horizontal padding
- Radius: 4px
- Hover: Darker background, slight lift

**Sections**
- Full-width container
- Alternating white/light-gray backgrounds
- Clear visual separation
- Consistent padding
