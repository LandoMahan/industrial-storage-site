- **Size:** <200KB compressed
- **Dimensions:** 1200x675px (standard)
- **Responsive:** Scales for mobile/desktop

## 10. Technical SEO

### robots.txt
Create `/robots.txt`:
```
User-agent: *
Allow: /
Disallow: /admin
Disallow: /api

Sitemap: https://serving-insured-jack-cherry.trycloudflare.com/sitemap.xml
```

### sitemap.xml
Create `/sitemap.xml` with all pages:
```xml
<?xml version="1.0" encoding="UTF-8"?>
<urlset xmlns="http://www.sitemaps.org/schemas/sitemap/0.9">
  <url>
    <loc>https://serving-insured-jack-cherry.trycloudflare.com/</loc>
    <lastmod>2026-04-08</lastmod>
    <priority>1.0</priority>
  </url>
  <url>
    <loc>https://serving-insured-jack-cherry.trycloudflare.com/pricing</loc>
    <lastmod>2026-04-08</lastmod>
    <priority>0.8</priority>
  </url>
  <!-- Add all pages -->
</urlset>
```

### Google Search Console
1. Go to https://search.google.com/search-console
2. Add property: `https://serving-insured-jack-cherry.trycloudflare.com`
3. Verify ownership (add DNS record or HTML file)
4. Submit sitemap.xml
5. Monitor:
   - Search keywords
   - Click-through rate
   - Impressions
