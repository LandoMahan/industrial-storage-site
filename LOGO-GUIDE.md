## Logo Design Elements

### Icon Specifications
- **Shape**: Warehouse building silhouette
- **Door**: Large industrial loading door (main feature)
- **Roof**: Triangular pitched roof with gold accent
- **Windows**: Small upper windows (professional appearance)
- **Details**: Door handles, foundation line, structural details
- **Color**: Blue gradient with gold accent
- **Style**: Clean, geometric, professional

### Logo Colors
| Element | Color | Hex | Purpose |
|---------|-------|-----|---------|
| Main Building | Blue Gradient | #0099cc → #0077aa | Primary identity |
| Accent | Gold | #ffdb58 | Emphasis, premium feel |
| Text | Dark Gray | #1a1a1a | Legibility |

### Logo Proportions
- **Icon aspect ratio**: 1:1 (square)
- **Lockup aspect ratio**: Approximately 3:1 (horizontal)
- **Minimum icon size**: 48px × 48px
- **Minimum lockup size**: 200px × 70px

## Usage Guidelines

### When to Use Each Logo

**Use Icon Only When**:
- Space is limited (favicon, small buttons)
- Logo appears at very small size (<100px)
- Social media profile picture
- App icon or bookmark
- Favicon (.ico)

**Use Full Lockup When**:
- Website header (desktop view)
- Email footer/signature
- Business documents
- Social media cover/banner
- Print materials (business cards, letterhead)
- Marketing collateral

### Size Specifications

| Usage | Minimum Size | Recommended Size |
|-------|--------------|------------------|
| Favicon | 16px | 32px - 64px |
| Social Profile | 200px | 400px - 800px |
| Website Header | 80px | 120px - 150px |
| Email Signature | 100px | 120px - 150px |
| Print (1" size) | 96 DPI | 300 DPI |

### Spacing (Clear Space)

**Minimum clear space**: 20px around entire logo
- Prevents crowding
- Maintains visual integrity
- No overlapping elements within this zone

## Color Variations

### Full Color (Primary)
- Blue gradient icon
- Dark text
- Gold accents
- **Use on**: White backgrounds, light backgrounds
- **Best for**: Digital, online, branding

### Single Color (Blue)
- Blue icon and text
- **Use on**: White/light backgrounds
- **Purpose**: When gradient isn't available
- **Hex**: #0099cc

### Reversed (White)
- White icon and text
- **Use on**: Dark backgrounds, photos, overlays
- **Purpose**: High contrast situations
- **Example**: Dark website sections, dark blue backgrounds

### Grayscale
- Icon and text in gray
- **Use on**: Black & white materials, specific applications
- **Hex**: #666666

## What NOT to Do

❌ **Don't**:
- Stretch or distort the logo
- Change the colors (without approval)
- Rotate the icon (keep horizontal)
- Remove parts of the design
- Add drop shadows or effects
- Place on busy/colorful backgrounds without contrast
- Use logo too small (below minimum size)
- Change the font or text styling
- Combine with competing logos
- Use outdated versions

✅ **Do**:
- Maintain aspect ratio
- Respect clear space
- Use official colors
- Check contrast on backgrounds
- Scale proportionally
- Test at actual display size
- Keep logo crisp and clean
- Use SVG for scalability
- Update designs when new version is released

## File Formats

### SVG (Preferred)
- **File**: `logo.svg`, `logo-lockup.svg`
- **Scalable**: Works at any size
- **Small file size**: Loads quickly
- **Editable**: Can modify colors, styles
- **Use for**: Web, digital, any screen-based use
- **Browser support**: All modern browsers

### PNG (Backup)
- **Use for**: Email, documents, non-web use
- **Transparency**: Yes (remove background if needed)
- **Sizes available**: 256px, 512px, 1024px
- **Note**: Generate from SVG as needed

### Favicon
- **Format**: .ico (Windows), .png (Modern browsers)
- **Size**: 16×16, 32×32, 64×64 pixels
- **Generation**: Use online favicon tool with logo.svg
- **Browser support**: All modern browsers
